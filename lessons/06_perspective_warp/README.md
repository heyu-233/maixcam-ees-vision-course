# 06 透视变换与中心误差

本节目标：
- 接着上一节已经排序好的四个角点，新增 `warp_perspective_target()`。
- 使用 `cv2.getPerspectiveTransform()` 和 `cv2.warpPerspective()` 把斜视角下的矩形标靶拉正。
- 直接取标准图中心作为目标点。
- 将标准图中心反变换回原图坐标。
- 计算目标点相对屏幕中心的 `err_x` 和 `err_y`。

运行方法：
1. 用 MaixVision 打开本目录。
2. 运行 `main.py`。
3. 对准矩形标靶，屏幕会显示真实四边形边框、目标中心点、屏幕中心点和误差信息。

这一节用于把“识别到矩形”推进到“得到可用于后续控制的目标误差”。
